# 🐍 Python Internship Projects

This repository contains the Python projects completed during my internship. Each project focuses on strengthening programming fundamentals, problem-solving, game development, and GUI development using Python.

## 📌 Project Overview

This repository consists of four Python-based applications developed as part of the internship program:

1. ⏰ Digital Clock
2. ❌⭕ Tic-Tac-Toe Game
3. 🐍 Snake Game
4. 🔴🟡 Connect Four Game

These projects demonstrate the practical application of Python programming concepts, including GUI development, event handling, game logic, data structures, and object-oriented programming.

---

# 📂 Project 1 – Digital Clock

## Description

A real-time digital clock application that displays the current system time in **HH:MM:SS** format. The clock updates every second automatically.

## Features

* Real-time clock
* Live time updates
* Digital display
* Simple and responsive interface

## Technologies Used

* Python
* Tkinter
* Time Module

---

# 📂 Project 2 – Tic-Tac-Toe Game

## Description

A classic two-player Tic-Tac-Toe game developed using Python. Players take turns placing **X** and **O** on a 3×3 grid until one player wins or the game ends in a draw.

## Features

* Two-player gameplay
* Win detection
* Draw detection
* Input validation
* Console-based implementation

## Technologies Used

* Python

---

# 📂 Project 3 – Snake Game

## Description

A classic Snake Game where the player controls a snake to collect food while avoiding collisions with walls and its own body.

## Features

* Keyboard controls
* Food generation
* Snake growth
* Collision detection
* Game Over screen
* Smooth gameplay

## Technologies Used

* Python
* Turtle Graphics
* Random Module
* Time Module

---

# 📂 Project 4 – Connect Four Game

## Description

A graphical implementation of the Connect Four game where two players compete to connect four discs horizontally, vertically, or diagonally.

## Features

* Interactive GUI
* Mouse-based controls
* Win detection
* Draw detection
* Player turn management
* Game over message

## Technologies Used

* Python
* Pygame
* NumPy
